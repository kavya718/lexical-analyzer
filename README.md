# 📝 Lexical Analyzer

A compiler design project built in **C** that scans source code and identifies lexical tokens such as keywords, identifiers, operators, literals, and special symbols. It performs lexical analysis by reading source files and classifying each token according to predefined language rules.

## ✨ Features

🔤 **Token Recognition** – Identifies keywords, identifiers, operators, constants, and symbols

📄 **Source Code Scanning** – Reads and analyzes source code files line by line

📊 **Token Classification** – Categorizes tokens based on lexical rules

🔢 **Constant Detection** – Recognizes numeric and character constants

❌ **Error Detection** – Detects invalid or unrecognized tokens

⚡ **Efficient Scanning** – Processes source code with optimized tokenization

🧩 **Modular Design** – Separate modules for scanning, token classification, and file handling
